Imprime depuis ton ordinateur 🖨️
Claudine, l'imprimante de l'Amicale, est dispo comme imprimante réseau. Ajoute-la à ton OS avec les infos ci-dessous, puis imprime depuis n'importe quelle app (Word, Pages, navigateur…). Les pages seront comptées et débitées automatiquement de tes crédits.
Tes identifiants
Adresse de l'imprimante (URL)
Nom d'utilisateur
Mot de passe (token d'impression)
Confidentiel — ce token donne accès à tes crédits d'impression. Si tu le tapes sur une machine partagée, pense à supprimer l'imprimante avant de partir, ou régénère ton token (en bas de page) pour invalider l'ancien.
Installation par appareil
1
Ouvre Réglages Système → Imprimantes & scanners → Ajouter une imprimante. La barre d'icônes par défaut ne propose pas l'option dont on a besoin (Avancé). Ctrl-clic (ou clic-droit) sur la barre d'icônes en haut de la fenêtre, puis choisis « Personnaliser la barre d'outils… » dans le menu qui apparaît.

La barre par défaut ne montre que les sources locales — il faut ajouter Avancé via Ctrl-clic.

2
Dans la palette qui s'ouvre, glisse l'icône « Advanced » (engrenage) vers la barre d'outils en haut de la fenêtre Add Printer. Clique Terminé.

Drag « Advanced » vers la barre d'outils, puis Terminé.

3
Clique sur la nouvelle icône Advanced (engrenage bleu) puis remplis :
  • Type : Internet Printing Protocol (ipps)
  • Device : Another Device
  • URL :
  • Name : Claudine
  • Location : Amicale du Wifi (libre)
  • Use : Generic PostScript Printer
Clique Add. macOS peut afficher « Unable to communicate » — c'est OK, clique Continue, l'imprimante se crée quand même.

Tous les champs remplis. URL = ipps://, jamais http(s)://.

4
Imprime un PDF de test (ex : Aperçu → Cmd-P → Claudine → Imprimer). macOS demande ton nom d'utilisateur (email) et ton mot de passe (token) en haut. Coche « Remember this password in my keychain » pour ne plus avoir à le retaper.

Entre ton email + ton token (les deux sont en haut de cette page).

5
Activer le recto-verso (duplex) : retourne dans Imprimantes & scanners → Claudine → Options & Supplies → Options, et bascule « Duplex Printing Unit » sur ON. Sans ça, la case « Recto-verso » disparaîtra du dialog d'impression. Clique OK.

Active le module Duplex pour que la case Recto-verso apparaisse dans le dialog d'impression.

6
C'est tout — la feuille sort à l'Amicale et tes crédits sont débités automatiquement. Les prochains prints se font sans demander de mot de passe (il est dans le trousseau).

5
Configurer les credentials sur le port — étape obligatoire Windows (équivalent du trousseau macOS, mais stocké sur le port plutôt que par session). Sans ça, le premier print échoue avec « Votre imprimante a rencontré un problème de configuration inattendu » (0x80070709) ou demande de configurer le port en boucle.
  • Clic-droit sur Claudine → Propriétés de l'imprimante
  • Onglet Ports → bouton Configurer le port…
  • Coche « Utiliser le compte d'utilisateur spécifié »
  • Nom d'utilisateur : ton email Amicale (en haut de cette page)
  • Mot de passe : ton token d'impression (en haut de cette page)
  • Clique OK. Les credentials sont mémorisés sur le port — tu n'auras plus à les saisir.
