Enregistrement de votre appareil
Votre appareil n'a pas été reconnu sur le réseau WiFi du coworking.
Pour activer la détection automatique de présence, suivez ces étapes.
1
Connectez votre laptop au réseau WiFi du coworking si ce n'est pas déjà fait.
2
Désactivez la randomisation MAC sur ce réseau :
macOS : Paramètres WiFi → détails du réseau → désactiver Adresse Wi-Fi privée
Windows : Paramètres → Réseau → WiFi → Propriétés → désactiver Adresses matérielles aléatoires

Reconnectez-vous à cette page depuis le WiFi du coworking. Votre appareil sera enregistré automatiquement.
