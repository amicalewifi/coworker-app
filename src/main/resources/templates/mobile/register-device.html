Détection automatique inactive
Votre appareil n'a pas pu être détecté automatiquement.
Activez la détection en suivant les étapes ci-dessous.
1
Connectez votre laptop au WiFi du coworking.
2
Désactivez la randomisation MAC sur ce réseau :
macOS : Réglages → WiFi → détails → désactiver Adresse Wi-Fi privée
Windows : Paramètres → Réseau → WiFi → Propriétés → désactiver Adresses matérielles aléatoires
3
Reconnectez-vous — votre appareil sera détecté automatiquement.
Ou saisir manuellement
Trouvez votre adresse MAC : macOS → Réglages WiFi → détails · Windows → ipconfig /all → "Adresse physique"
